This project presents a visual and interactive analysis of a **PCOS case study dataset (465 participants)**.  
It highlights key patterns related to **diagnosis rates, symptom distribution, menstrual health, lifestyle habits, and demographic breakdowns**.

The dashboard includes sections for:

- **Overview of Participants** (age groups, blood type, weight vs height)  
- **Diagnosis Insights** (symptoms like acne, hair loss, mood swings, fast-food habits, exercise patterns)  
- **Symptom Analysis** (comparisons of PCOS vs non-PCOS groups)  
- **Menstrual Health** (cycle duration, frequency, regularity)  
- **Lifestyle Correlations** (fast food, exercise, blood type associations)

This project provides a concise yet meaningful understanding of PCOS-related patterns using clear charts and structured visuals.
